# EventReg - Event Registration System

A web-based event registration system built with HTML, CSS, JavaScript, and PHP.

## Features
- Browse available events
- Register for events online
- Contact form
- FAQ page
- Admin panel to view registrations

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP (localhost)

## Pages
- Home, About, Events, Register
- Contact, FAQ, Registration Success
- Admin: View Registrations

## Setup
1. Clone the repo
2. Place in XAMPP htdocs folder
3. Create database and import SQL
4. Update config.php with your DB credentials
5. Run via localhost
